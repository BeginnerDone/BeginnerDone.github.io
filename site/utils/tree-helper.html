Tree-List互转

本页内容

介绍 ​

常用于tree的常用操作

方法 ​

listToTree ​

集合转为树结构，通过对应的parentId进行转换

示例

ts
 const list = [
        { id: 1, parentId: null },
        { id: 2, parentId: 1 },
        { id: 3, parentId: 1 },
        { id: 4, parentId: 2 },
        { id: 5, parentId: 2 }
    ];

   listToTree(list,{
       id: "id",
       pid: "parentId",
       children: "children"
   })

  // 最后输出 tree
  [
        {
            id: 1,
            parentId: null,
            children: [
                {
                    id: 2,
                    parentId: 1,
                    children: [
                        { id: 4, parentId: 2 },
                        { id: 5, parentId: 2 }
                    ]
                },
                { id: 3, parentId: 1 }
            ]
        }
   ];

treeToList ​

将树结构转为集合，通过parentId

ts
 const tree = [
        {
            id: 1,
            parentId: null,
            children: [
                {
                    id: 2,
                    parentId: 1,
                    children: [
                        { id: 4, parentId: 2 },
                        { id: 5, parentId: 2 }
                    ]
                },
                { id: 3, parentId: 1 }
            ]
        }
   ];
   treeToList(list,{
       id: "id",
       pid: "parentId",
       children: "children"
   })

  // 最后输出 list
   [
        { id: 1, parentId: null },
        { id: 2, parentId: 1 },
        { id: 3, parentId: 1 },
        { id: 4, parentId: 2 },
        { id: 5, parentId: 2 }
    ];

findNodeAll ​

用于在树形结构中查找所有满足特定条件的节点的集合。

ts
 const tree = [
        {
            id: 1,
            parentId: null,
            children: [
                {
                    id: 2,
                    parentId: 1,
                    children: [
                        { id: 4, parentId: 2 },
                        { id: 5, parentId: 2 }
                    ]
                },
                { id: 3, parentId: 1 }
            ]
        }
   ];
findNodeAll(tree, (node: any) => node.id > 2,{
       id: "id",
       pid: "parentId",
       children: "children"
 })
// 输出
【{ id: 4, parentId: 2 },{ id: 5, parentId: 2 }, { id: 3, parentId: 1 }】

findPath ​

在树中找到满足给定条件的节点的路径。

ts
/**
 * 在树中找到满足给定条件的节点的路径。
 * @example ["root", "child1", "child2"]
 * @template T - 树中节点的类型。
 * @param {any} tree - 要搜索的树。
 * @param {(node: any) => boolean} func - 节点必须满足的条件。
 * @param {Partial<TreeOption>} [config={}] - 树遍历的配置选项。
 * @returns {T | T[] | null} - 满足条件的节点的路径，如果未找到则返回 null。
 */
