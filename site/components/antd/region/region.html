介绍 ​

主要用于行政区域的级联选择，当然该组件同样支持符合行政区域的这种级联的数据结构，只需要配置好fieldNames即可，

基础 ​

region
行政区域
<template>
    <w-region v-model:level-options="levelOption" @change="change" />
</template>
<script lang="ts" setup>
import { reactive, ref } from "vue";
import type { LevelOptions } from "wlink-ui/antd";

const levelOption = reactive<LevelOptions[]>([
    {
        title: "省",
        code: "",
        name: "", // 非必填
        data: []
    },
    {
        title: "市",
        code: "",
        name: "", // 非必填
        data: []
    }
]);

const provinceOptions = ref([
    {
        name: "广东省",
        code: "440000"
    },
    {
        name: "安徽省",
        code: "340000"
    }
]);

const ah = [
    {
        name: "合肥市",
        code: "340100"
    },
    {
        name: "芜湖市",
        code: "340200"
    },
    {
        name: "蚌埠市",
        code: "340300"
    },
    {
        name: "淮南市",
        code: "340400"
    }
];
const gd = [
    {
        name: "广州市",
        code: "440100"
    },
    {
        name: "深圳市",
        code: "440300"
    },
    {
        name: "珠海市",
        code: "440400"
    }
];

const change = ({ item, index }) => {
    if (index === 0) {
        // 选择省,加载市
        if (item.code === "440000") {
            levelOption[1].data = gd;
        } else if (item.code === "340000") {
            levelOption[1].data = ah;
        } else {
            levelOption[1].data = [];
        }
    } else if (index === 1) {
        // 选择市
    }
};

const addData = () => {
    // 加载省数据
    levelOption[0].data = provinceOptions.value;
};
addData();
</script>

    <w-region
        ref="regionRef"
        v-model:level-options="levelOption"
        @change="change"

属性 ​

参数说明类型默认值
fieldNames行政区域集合主要字段，code标识行政区域的编码唯一标识，name一般标识显示名称Object{ name: "name", code:"code" }
levelOptions(v-model)数据集合，具体请查看配置项LevelOptions[]
popover继承气泡卡片 Popover - Ant Design Vue (antdv.com)

levelOptions配置项 ​

参数说明类型默认值
title显示的标题，默认tab选项卡的标题String
code选中的行政区域编码String
name选中的行政区域名称String
data所属行政级别的数据集合Array

插槽 ​

方法名描述参数
tab顶部选项卡插槽{record:LevelOptions，index:number}
option行政区域列表项的插槽{record:any}

方法|属性 ​

方法名描述参数
getTitle获取tab标题的方法(level:LevelOptions)
selectName获取选中的数据/ 分割的名称字符串

事件 ​

事件名说明回调
change选中列表项触发({item: any,level:LevelOptions,index:number}) ` 其中item表示列表项的数据，level表示某一个tab的数据，index表示tab所在的下标
