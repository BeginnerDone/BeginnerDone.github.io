介绍 ​

继承了 a-tree组件除slot外所有的属性

基础 ​

No data

选择结果：

tree
树形控件
<template>
    <w-tree :tree-data="options" placeholder="请选择" @select="onSelect" />
    <span> 选择结果：{{ modelValue }} </span>
</template>
<script lang="ts" setup>
import { ref } from "vue";
const modelValue = ref();
const options = ref([
    {
        name: "测试1",
        value: "1",
        children: [
            {
                name: "测试1-1",
                value: "1-1"
            }
        ]
    },
    {
        name: "测试2",
        value: "2",
        children: [
            {
                name: "测试2-1",
                value: "2-1"
            }
        ]
    }
]);

const onSelect = value => {
    modelValue.value = value;
};
</script>

远处加载 ​

No data

url
设置url属性动态加载下拉数据，reload方法重新加载
<template>
    <a-button @click="reload">重新加载</a-button>
    <w-tree
        ref="treeRef"
        url="tree/list"
        placeholder="请选择"
    />
</template>
<script lang="ts" setup>
import { ref } from "vue";

const treeRef = ref<any>(null);

function reload() {
    treeRef.value.reload();
}
</script>

搜索过滤 ​

No data

选择结果：

filter
设置filter属性，开启搜索框过滤树
<template>
    <div style="height: 200px">
        <w-tree
            ref="treeRef"
            filter
            url="tree/list"
            placeholder="请选择"
            resize
            @select="onSelect"
        />
    </div>
    <span> 选择结果：{{ modelValue }} </span>
    <a-button @click="reload">重新加载</a-button>
</template>
<script lang="ts" setup>
import { ref } from "vue";
const modelValue = ref();
const treeRef = ref();
const reload = () => {
    treeRef.value.reload();
    modelValue.value = undefined;
};

const onSelect = value => {
    modelValue.value = value;
};
</script>

属性 ​

参数说明类型默认值
url动态加载 tree数据。string
method动态加载 tree数据的请求方法stringpost
isBody动态加载 tree数据时，是否以body携带参数Booleanfalse
params动态加载 tree数据时，请求参数Object
fieldNames替换 tree中 title,key,children 字段为 tree中对应的字段Object {children:'children', title:'name', key: 'value' }
filter是否开启搜索框Booleanfalse
resize根据父容器高度设置虚拟滚动高度Booleanfalse

事件 ​

事件名说明回调
onComplete远程加载数据成功response
onError远程加载数据失败error
