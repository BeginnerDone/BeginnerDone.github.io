融合和 a-form组件以及antdv中其他的表单组件形成的搜素表头，该组件主要是为了a-table服务组成业务组件，当然单独使用也是可以的

基本的搜素
通过设置基本的items属性即可生成一个搜索表单
<template>
    <div>
        <w-form-search :items="items" @search="onSearch" />
        <div>
            结果：
            <pre>{{ form }}</pre>
        </div>
    </div>
</template>
<script lang="ts" setup>
import { ref } from "vue";
const items = ref([
    {
        type: "input",
        itemAttrs: {
            label: "姓名",
            name: "name"
        }
    },
    {
        type: "input",
        itemAttrs: {
            label: "年龄",
            name: "age"
        }
    },
    {
        type: "inputNumber",
        itemAttrs: {
            label: "分数",
            name: "score"
        }
    },
    {
        type: "select",
        autoSearch: true,
        itemAttrs: {
            label: "班主任",
            name: "teacher"
        },
        componentAttrs: {
            url: "select/list"
        }
    },
    {
        type: "datePicker",
        itemAttrs: {
            label: "日期",
            name: "date"
        },
        dateTimeAttrs: {
            range: true,
            startField: "startDate",
            endField: "endDate"
        }
    }
]);
const form = ref({
    name: "888"
});

const onSearch = (value: any) => {
    form.value = value;
};
</script>

initValue和model
通过设置initValue或者model属性可以设置默认的查询条件，切记这里最好是使用initValue去绑定，因为model属性主要是为了动态改变一个值，这里initValue表示默认查询条件不会随着重置而改变，所以如果要重新回到空的状态，那么使用model绑定初始化是最好的办法
<template>
    <div>
        <a-divider orientation="left">设置items中的initValue属性</a-divider>
        <w-form-search :items="items" @search="onSearch" />
        <a-divider orientation="left">设置model属性</a-divider>
        <w-form-search
            v-model:model="form"
            :items="items1"
            @search="onSearch"
        />
    </div>
</template>

属性 ​

整体属性 ​

参数说明类型默认值
items表单项，配置项ExtFormItemp[][]
formAttrsForm表单属性Object
v-model (model)表单数据Object
actionSpan操作按钮占据宽度，通常配合items中的span进行配置Number6
span每个form-item默认的宽度Number6

items属性 ​

参数说明类型默认值
type表单项类型，这里支持input, select, datePicker, radio, checkbox, treeSelect, switch, inputNumber, timePicker。 同时对于datePicker和timePicker的查询条件，如果存在initValue且非空，注意增加componentAttrs的valueFormat属性去格式化数据，其中。select使用w-select组件，datePickerrange模式（dateTimePicker中range为true）则使用w-date-range-picker组件。treeSelect使用w-tree-select组件Stringinput
span所有输入表单项占据的宽度Number6
initValue搜索条件默认值 (对于日期区间而言就是[开始时间,结束时间] 的数组格式)，初始化，重置之后回显示该值。any
autoSearch是否自动触发搜索，比如日期区间选择后自动触发搜索这种需求Booleanfalse
componentAttrs各表单组件自有和扩展属性Object
itemAttrsform-item自有属性Object
dateTimeAttrs针对日期|时间组件特有的属性，配置项

dateTimeAttrs属性 ​

参数说明类型默认值
range是否时间区间Booleanfalse
startField开始字段String
endField结束字段String
